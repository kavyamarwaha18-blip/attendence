# 📋 Attendance Tracker

A simple and clean **Attendance Tracker** web application built as a Cloud Computing subject project.

## 🖥️ Live Demo

> Upload to GitHub Pages to get a live link:
> `https://YOUR-USERNAME.github.io/attendance-tracker/attendance.html`

---

## 📌 Features

- ✅ Mark students as **Present**, **Absent**, or **Late**
- 📊 Live **stats dashboard** (Total, Present, Absent)
- 📈 **Attendance percentage** progress bar
- 🔍 **Filter** by subject, status, or search by name/roll number
- 🔄 **Toggle** attendance status with one click
- 🗑️ Delete individual records or clear all
- ⬇️ **Export to CSV** for download
- 💾 Data saved in **browser localStorage** (no backend needed)

## 🚀 How to Run

### Option 1: Open Locally
Just double-click `attendance.html` — it opens directly in any browser. No setup needed.

### Option 2: GitHub Pages (Free Hosting)
1. Upload this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch**
4. Your site will be live at `https://YOUR-USERNAME.github.io/attendance-tracker/`
